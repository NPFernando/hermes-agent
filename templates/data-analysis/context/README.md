# Data Analysis Project Template

## Overview
This template sets up a Python data analysis project with Jupyter notebooks,
pandas, numpy, and visualization libraries pre-configured.

## Quick Start

### 1. Install Dependencies
```bash
pip install pandas numpy matplotlib seaborn jupyter scikit-learn
```

### 2. Start Jupyter
```bash
jupyter notebook
```

### 3. Project Structure
```
project/
├── notebooks/       # Jupyter notebooks
├── data/            # Datasets
├── reports/         # Generated reports and figures
└── src/             # Reusable Python modules
```

## Included Skills
- **data-science**: Data exploration, analysis, and visualization workflows
- **jupyter-live-kernel**: Iterative Python via live Jupyter kernel

## Recommended Workflow
1. Place raw data in `data/`
2. Create exploration notebooks in `notebooks/`
3. Extract reusable code to `src/`
4. Generate reports and figures in `reports/`
